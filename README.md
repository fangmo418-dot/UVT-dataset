# UVT: Underwater Video-Text Dataset

## Overview
UVT is a multimodal underwater video dataset for multi-label classification.  
It contains 4,773 short video clips, structured English scene descriptions, multi-label annotations, and data split files.

The dataset is designed to support research on underwater video understanding, multimodal learning, and multi-label classification under complex visual degradation conditions.

## Release Status
The UVT dataset is currently being organized and curated for public release.  
Due to the large storage size of the video data and the need for further organization and documentation, the full dataset is not yet publicly available.

The complete dataset, including video files, annotations, metadata, and split files, is planned to be released after the related paper is formally published.

## Contents
The UVT dataset includes the following components:

- Underwater video clips
- Structured English scene descriptions
- Multi-label annotation files
- Train/test split files
- Sample subsets for preview and testing
