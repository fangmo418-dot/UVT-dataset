# UVT: Underwater Video-Text Dataset

## Overview
UVT is a multimodal underwater video dataset for multi-label classification.  
It contains 4,773 short video clips, structured English scene descriptions, and multi-label annotations.

The dataset is designed to support research on underwater video understanding, multimodal learning, and multi-label classification under complex visual degradation conditions.

## Release Status
The UVT dataset is currently being organized and curated for public release.  
Due to the large storage size of the video data and the need for further organization and documentation, the dataset will be released progressively, and additional data and related materials will be updated continuously in this repository.

The complete dataset, including video files, annotation files, and related metadata, will be made available after the related paper is formally published.

To facilitate preliminary preview and testing, six sample videos have been provided in the `sample` folder of this repository.

## Contents
The UVT dataset currently includes or will include the following components:

- Underwater video clips
- Structured English scene descriptions
- Multi-label annotation files
- Sample data for preview and testing
